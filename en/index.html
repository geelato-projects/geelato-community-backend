Geelato Project

More than documentation, this is a knowledge base for developers and AI to build complex business systems together.
The project is built on three core pillars: Geelato Framework for foundational architecture and modeling, AuthServer for unified authentication, and Message for unified messaging.

Core Capability Matrix
MQL & ORM

Metadata query language and Fluent DSL with seamless support for major relational databases such as MySQL, PostgreSQL, and Oracle.

Runtime & Designer

Separates runtime and design-time responsibilities to support hybrid low-code and pro-code development.

Auth Server

An independent authentication center that provides standard OAuth2 support and lightweight lite-login integration.

Plugin & SPI

Provides replaceable platform-level extension points around Meta, Security, and ORM.
