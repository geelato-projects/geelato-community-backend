# SrvExplain API Catalog

This page lists the API documentation currently generated under `SrvExplain/` and exposes it as a static API catalog entry inside the official documentation site.

## How to Use This Page

- this page lists the controller documents currently generated in `SrvExplain`
- each link points to the corresponding markdown file in the repository
- for request methods, paths, parameters, and examples, continue into each controller document
- when `SrvExplain` is regenerated, this catalog should be updated accordingly
