# PlatformWebRuntime

`PlatformWebRuntime` 是 Web 平台拆分后面向运行时的应用层。

## 定位

它代表“业务执行所需的最小可运行子集”，主要承接：

- 运行时 HTTP 接口
- 运行时部署链路
- 面向业务执行的基础能力
- 经过筛选的运维能力

## 典型归属能力

- 运行时鉴权入口
- 文件与附件访问能力
- 消息、通知、解析、OCR、AI、SSE 等运行时接口
- 共享运行时基础设施与启动链路

## 不应进入运行时最小子集的能力

- 元数据定义编辑
- 模型与表设计
- 脚本设计与管理
- 打包发布能力
- 偏后台治理的设计时接口

## 核心原则

运行时应用应当可以在不依赖完整设计时能力的情况下独立部署。
