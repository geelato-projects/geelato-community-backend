# PlatformDesginer

`PlatformDesginer` 是 Web 平台拆分后面向设计时的应用层。

## 定位

它在运行时子集之上，继续承接设计时配置、建模和治理能力。

## 典型设计时范围

- 元数据管理
- 模型与表设计
- 脚本与 API 定义管理
- 打包与发布能力
- 偏管理侧的安全与治理接口

## 与运行时的关系

- 设计时侧包含运行时基线能力
- 运行时侧不能反向依赖设计时编辑与治理能力

## 为什么要进入官方文档

这个拆分已经不再只是代码结构问题，而是直接影响：

- 模块边界
- 部署模式
- API 分组
- 后续 OpenAPI 治理方式
